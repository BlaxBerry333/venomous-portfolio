## 担当役割と成果

- フロントエンドの再構築と技術スタック選定をリードし、**Vite + React + SWC** によるホットリロード時間を平均 8s から 1s 未満に短縮するとともに、サーバー起動時のコールドスタートも従来より大幅に改善。チーム全体の開発体験を向上させ、生産性を高めた
- ノードベースの可視化ワークフローシステムをゼロから設計・構築。従来は開発チームが手作業で行っていた設定フローを置き換え、営業担当者が数分〜数十分で顧客設定を完了できるようにし、導入リードタイムを大幅に短縮
- メディアリソースの統合管理機能を提案し、**Django + Django REST Framework** を用いてバックエンド API を設計・構築。社内オペレーションチームの業務効率を大幅に向上
- 既存 **Go + Gin + Protocol Buffers** のマイクロサービスと **Node.js** の BFF を利用した API 開発を担当し、機能実装と既存コードの保守・改善を行った
- プロダクトチームと密に連携し、ユーザーフィードバックをもとに改善サイクルを設計・推進。入社後は四半期ごとの OKR 評価で一貫して最高評価 ( A+ ) を獲得
