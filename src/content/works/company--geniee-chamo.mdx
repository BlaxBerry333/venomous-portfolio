## 担当役割と成果

- フロントエンドの再構築と技術スタック選定をリードし、**Vite + React + SWC** によるホットリロード時間を平均 8s から 1s 未満に短縮。開発体験を大幅に改善し、チーム全体の生産性を向上
- **Go + Gin** のマイクロサービスと **Protocol Buffers** を導入し、高負荷なチャット処理を分離
- ノードベースの可視化ワークフローシステムをゼロから設計・構築。社内での利用導入により、従来開発チームが数日かけていた顧客ごとの設定作業を営業担当者が数十分で完了可能になり、導入リードタイムを大幅に短縮
- 提案了 Media 资源的统一管理功能并主导设计与用**Django + Django REST Framework + S3** を用いてバックエンド API を構築，进一步提高了社内运营团队的效率
- プロダクトチームと密に連携し、ユーザーフィードバックに基づいた A/B テストを推進し、顧客満足度を向上
