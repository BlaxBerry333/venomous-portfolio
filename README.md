## Features

- Multi-theme architecture (Nebula / Manga)
- Multi-language support (日本語 / English / 中文)
- Responsive layout with mobile-first approach
- Page transition animations with View Transitions API
- WebGL shader effects and interactive 3D scenes
- SEO optimized with proper meta tags
